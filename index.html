Editing Tools

  • Remove Background: Click on the color to make transparent. Adjust tolerance for similar colors.
  • Crop: Select an area to keep. Use presets for common ratios.
  • Rotate: Rotate 90¬∞, 180¬∞, or any custom angle. Flip horizontally/vertically.
  • Resize: Change dimensions by pixels or percentage. Lock aspect ratio to prevent distortion.

Pro Tips

  • Use PNG for images with transparency
  • Use JPG for photos (smaller file size)
  • Use WebP for modern websites (best compression)
  • ICO format supports multiple sizes - great for favicons
  • Higher quality = larger file size
  • Batch process multiple files at once
